layer_a.dat
=============

.. Note:: ``options.calc_vert_profiles`` & ``options.calc_ebal``

rows - time (simulation number)

columns - photosynthesis per layer, total soil respiration (60 + 1)

.. list-table::
    :widths: 20 20 60

    * - variable
      - units
      - description
    * - **A1d**
      - umol m-2 s-1
      - mean photosynthesis leaves, per layer
    * - **Resp**
      - umol m-2 s-1
      - soil respiration rate
